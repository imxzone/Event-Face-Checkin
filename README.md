## Description
A real-time event check-in system using AI-based face recognition from a webcam, optimized for Edge (CPU) environments.

## Workflow
- Collect and preprocess face images  
- Extract embeddings and build a face database  
- Perform real-time face recognition via webcam  
- Match and confirm identities  
- Log check-in information automatically
